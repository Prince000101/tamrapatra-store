# Tamrapatra Backend

## Install

```bash
npm install
```

## Create .env

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

## Run Backend

```bash
npm run server
```

## Seed Database

```bash
npm run seed
```

## API Routes

### Get All Products

```bash
GET /api/products
```

### Get Single Product

```bash
GET /api/products/:id
```
